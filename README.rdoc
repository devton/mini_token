= MiniToken

Simple token generator for short url and some other things :)

== Instalation

	gem install mini_token

== Example

Generate token with default size
	
	MiniToken.output
	# => "S8hQNy"

Generate token with other size
	
	MiniToken.output(9)
	# => "2u27Jum8Y"
